# HTA startup persistence

Use this query to locate persistence in Startup with HTA files.

## Query
```
DeviceFileEvents
| where FolderPath contains @"\AppData\Roaming\Microsoft\Windows\Start Menu\Programs\Startup\"
| where FileName endswith ".hta"
```
